Canonical Navigation Banner

A very simple example of the Canonical Navigation Banner to facilitate screenshots of it.

Instructions:

  1. Edit the text in the simple-nav-banner.html file to your liking:
    2. <span class="p-navigation__logo-title">
  &nbsp;Your Text Here
</span>
  2. Refresh the page
  3. Take a screenshot of just the navigation bar inside the dashed border
  4. Crop the image if needed to include only the dark navigation bar
  5. Use the image wherever you'd like!
  6. N.B. You can also change the width by resizing your browser window
